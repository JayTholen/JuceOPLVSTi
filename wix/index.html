FM synthesizer VST

Yamaha OPL chip sounds wrapped in a VST. Create classic DOS game sounds in modern DAWs. As heard in late 80s / early 90s PC sound cards. DOSBox OPL emulation under the hood.

Builds

 

I only have access to a Windows machine. The code builds on OSX (XCode) and Linux thanks to the magic of Juce. Give back by building when new versions are released.

Presets

 

Make it more fun for everyone. I've collected hundreds of .SBI presets from various sources, including user contributions. Rip from your favourite DOS games or export from the plugin itself with one click. No excuses!

Bug fixes and reports

 

I'm doing my best to keep on top of serious issues, but there a lot of combinations of operating system and software. Any and all help is greatly appreciated.

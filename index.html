JuceOPLVSTi

A VST instrument using the DOSBox OPL2 emulation

Download this project as a .zip file Download this project as a tar.gz file

The OPL2 is an 2-operator digital FM chip from the late 80s, used in PC sound cards like the Adlib and original Sound Blaster.
